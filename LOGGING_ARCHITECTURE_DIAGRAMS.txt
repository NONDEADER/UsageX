## Data Flow: New Log Entry

  Component Code              RawLog Function            Storage
  ──────────────────          ───────────────            ───────
       │
       ├─→ await Log("warn", "Something happened")
       │                            ↓
       │                  Parse level: "warn"
       │                  Check debug_mode_until
       │                  console.warn() output
       │                  Generate timestamp
       │                  Serialize arguments
       │                  Create logEntry object
       │                            ↓
       │                  storage.local.get('debug_logs')
       │                            ↓
       │                  logs = [] or [existing...]
       │                  logs.push(newEntry)
       │                  
       │                  if (logs.length > 1000)
       │                    logs.shift()  ← Remove oldest
       │                            ↓
       │                  storage.local.set({
       │                    debug_logs: logs
       │                  })
       │                            ✓ Saved


## Auto-Refresh Mechanism

  Debug Page Loads
       ↓
  showLogs()  ────→ Fetch and render all logs
  updateDebugStatus()  ────→ Update countdown timer
  startAutoRefresh()
       ↓
  setInterval(5000ms)
       ├─→ if (autoUpdate checkbox) {
       │     showLogs()  ← Full re-render
       │   }
       │
       ├─→ updateDebugStatus()  ← Update countdown
       │
       └─→ Loop every 5 seconds
       
  When debug mode expires:
       ├─→ stopAutoRefresh()
       └─→ Display "Debug mode disabled"

## Log Entry Lifecycle

Created              Displayed            Auto-refresh         Cleared
  │                    │                     │                   │
  ├─ timestamp       ├─ colorized          ├─ 5s loop         ├─ user
  ├─ sender          ├─ DOM spans          ├─ full rerender   │  clicks
  ├─ level           ├─ auto-scroll        ├─ continuous      │  Clear
  ├─ message         └─ in viewer          └─ until expired   │
  │                                                           ├─ All
  ├─ Serialized                                              │  entries
  │  (errors,                                                │  removed
  │   objects)
  │
  ├─ Storage (1000 max)
  │  FIFO eviction
  │  if > 1000:
  │    shift()
  │
  └─ Persists until
     cleared or
     extension
     uninstalled

## Message Flow: Log Viewer Refresh

User clicks "Refresh"
        │
        ▼
showLogs()
        │
        ├─→ browser.storage.local.get('debug_logs')
        │
        ├─→ Filter logs array [log1, log2, ...]
        │
        ├─→ Clear DOM: preElement.innerHTML = ''
        │
        ├─→ For each log:
        │   ├─ Create div.log-line
        │   ├─ Set data-level attribute
        │   ├─ Create span.log-timestamp
        │   ├─ Create span.log-sender
        │   ├─ Create span.log-message
        │   └─ Append to container
        │
        ├─→ scrollToBottom()
        │   └─ preElement.scrollTop = scrollHeight
        │
        ▼
     Display Updated


## Error Handling in Logging

┌──────────────────────────────────────────┐
│    ERROR PATHS THAT GET LOGGED           │
├──────────────────────────────────────────┤
│                                          │
│ 1. Explicit Error Logging                │
│    └─ await Log("error", errorObj)      │
│    └─ Caught in try/catch blocks        │
│                                          │
│ 2. Global Error Handler                  │
│    └─ window.addEventListener('error')  │
│    └─ Auto-logs via logError()          │
│                                          │
│ 3. Unhandled Promise Rejection           │
│    └─ addEventListener('unhandledrejection')
│    └─ Auto-logs via logError()          │
│                                          │
│ 4. Error Serialization                   │
│    └─ Stack trace extracted             │
│    └─ error.stack preserved             │
│    └─ Fallback to toString()            │
│                                          │
└──────────────────────────────────────────┘
