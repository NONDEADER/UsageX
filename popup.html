What is UsageX?

UsageX is a browser extension that tracks your Claude.ai token usage in real time. It injects a live stats panel directly into the Claude page — no external server, no third-party services, all data stays on your device.

Dashboard
Session bar (5 h)

Shows how much of your rolling 5-hour token window you've consumed. Hover the bar to see raw token counts.

Weekly bar

Tracks usage across your 7-day token window. Resets on the same rolling schedule as Claude's own limits.

Rate indicator dot

The small pulsing dot next to the session % shows your current burn rate:

Teal — below normal (<19%/h) — great pace Gray — on track (~20%/h) Amber — above normal (>21%/h) Orange — burning fast (≥30%/h) — slow down!
Live dot (header)

Teal = off-peak hours. Red / fast pulse = peak traffic window (6:30 PM – 12:30 AM IST on weekdays), when Claude limits deplete faster.

Sidebar Panel (on Claude)
Stats cards

Messages sent, conversations started, active time, and estimated token cost for today's session — all updated live after each message.

Peak-hours clock

A 24-hour timeline strip highlighting high-traffic hours (6:30 PM – 12:30 AM IST). During this window Claude's shared infrastructure is busiest and limits can deplete quicker.

Messages remaining

Estimated messages left in the current 5-hour window, derived from your average token cost per message this session.

Settings (⚙ in sidebar)
Sidebar side

Dock the panel on the left or right of the Claude navigation sidebar.

Floating mode

Detaches the panel so it floats freely anywhere on screen. Drag it by the header to reposition.

Opacity

Controls floating-panel transparency (10–100%). Useful for keeping the stats visible without blocking the page.

Resizable

Enables drag handles so you can resize the floating panel to any width or height.

Timezone

Override the timezone used for reset-time display. Defaults to your browser's local timezone.

Debug logging

Toggle verbose event logging. View logs in the Tools tab → Debug Logs, or export them as JSON.

Keyboard Shortcuts
Toggle sidebar panel Alt+U
About Token Estimates
Token counts are approximate and labelled ~est in the UI. Claude doesn't expose exact counts in the browser; input length is estimated from character count (chars ÷ 4) and thinking tokens are estimated per effort level.
Privacy
All data is stored in browser.storage.local — on your device only.
UsageX makes zero network calls of its own.
No third-party libraries, no analytics, no telemetry.
