Privacy Policy

Last updated: July 2026  ·  Applies to UsageX v2 (Firefox & Chrome extension)

Short version: UsageX is 100% private and runs entirely locally. It does not transmit any user data, usage statistics, or message contents to external servers.

1. What data is collected

UsageX tracks your Claude.ai usage statistics and stores them entirely within your browser's local storage (browser.storage.local and IndexedDB). No account is created, no server is involved, and no data is synced to any cloud service.

Data stored locally includes:

  • Session and weekly token usage percentages (read from Claude's own API responses)
  • Daily message count, active time, estimated token totals
  • Effort breakdown (Low / Medium / High / Max thinking levels)
  • Models used per session
  • Conversation-level message and token estimates
  • Reset timestamps for session and weekly limits
  • Your Claude plan (Free / Pro / Max), name, and email — scraped from the Claude UI or API response and used only to label the dashboard
  • A rolling window of one-way djb2 hashes of outgoing messages, used exclusively to prevent double-counting (the original message text is never stored)
  • Extension settings (sidebar position, notification preferences, timezone, etc.)

2. Data transmission

No data leaves your device. UsageX makes zero network calls and does not communicate with any external servers or third-party services. All processing and storage occur entirely within your web browser.

3. Permissions — why each is needed

PermissionWhy it's needed
storageStores all usage stats and settings locally on your device
alarmsSchedules local notifications for session/weekly limit resets
tabsQueries active tab context to detect Claude.ai in the popup, focuses existing Claude tabs, and opens the debug viewer
notificationsShows browser-native reset alerts — only if you enable this in Settings
host: claude.aiAllows the content script and fetch hook to read usage data from Claude's API responses

4. Data retention and deletion

All locally stored data can be deleted at any time from within the extension:

  • Reset Stats — clears today's counters and the 30-day history
  • Clear Logs — deletes all debug log entries
  • Uninstalling the extension removes all local storage automatically

5. Third-party services

UsageX does not use any analytics services, crash reporting tools, advertising networks, or external cloud backends. The extension is completely offline.

6. Children

UsageX does not collect or transmit information from anyone, including children under 13.

7. Changes to this policy

If the privacy practices of UsageX change significantly, this page will be updated and the "Last updated" date will change. Continued use of the extension constitutes acceptance of the updated policy.
